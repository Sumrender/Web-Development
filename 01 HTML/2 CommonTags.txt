First you should know the basics html, head, and body tags
which make for the structure of the webpage.
Refer the atom html Boilerplate for more details, it is more or
less intuitive.
<html></html> : tells, inside is html code

<head></head> : consists of title, icons, meta and all major links, CDNs

<body></body> : contains all the main objects which appear on the webpage.

The <h1> to <h6> HTML elements
    Heading element
    requires ending tag: <h1>yo</h1>
    Represent six levels of section headings.
    <h1> is the highest section level and <h6> is the lowest.
<br>
    line break element
    self closing tag

<span>YO</span>
    The <span> tag is an inline container used to mark up a part of a text, or a part of a document.
    The <span> tag is easily styled by CSS or manipulated with JavaScript using the class or id attribute.
    The <span> tag is much like the <div> element, but <div> is a block-level element and <span> is an inline element.

<hr>
    horizontal rule element
    used to add a horizontal line

<center>
    centers the Text
    requires closing tag

<p></p>
    paragraph tag
    
<em> or <i> tag - em tag preferred
    requires closing tags
    for emphasis on the Text

<b> or <strong> tag - strong tag preferred
    requires closing tag
    for strong importance
